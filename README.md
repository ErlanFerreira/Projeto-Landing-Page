# Projeto Landing Page

Este projeto consiste no desenvolvimento de uma **Landing Page** criada como exercício prático durante o curso de tecnologia da **Escola DNC**.

O objetivo foi aplicar conceitos fundamentais de **desenvolvimento web**, utilizando as tecnologias base da web para construir uma página moderna, responsiva e funcional.

---

## Sobre o Projeto

A Landing Page foi desenvolvida com foco em:

* Estruturação de páginas web
* Estilização e layout responsivo
* Interatividade com JavaScript
* Organização de código para projetos web

Este projeto faz parte das atividades práticas propostas no curso, visando consolidar o aprendizado em desenvolvimento front-end.

---

## Tecnologias Utilizadas

As seguintes tecnologias foram utilizadas no desenvolvimento:

* **HTML5** – Estrutura da página
* **CSS3** – Estilização e layout
* **JavaScript** – Interatividade e funcionalidades da página

## Como Executar o Projeto

1. Clone este repositório:

```
git clone https://github.com/ErlanFerreira/Projeto-Landing-Page.git
```

2. Acesse a pasta do projeto:

```
cd Projeto-Landing-Page
```

3. Abra o arquivo **index.html** no navegador.
